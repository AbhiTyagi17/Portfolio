## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/portfolio.git
```

Navigate to the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run Tailwind CLI:

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./dist/output.css --watch
```
